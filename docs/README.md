# Organização dos materiais

- `trabalho/`: documentação técnica, teste de mesa e guia de datasets para aprofundar o algoritmo.
- `apresentacao/`: materiais voltados à comunicação do projeto (slides, roteiro, referências para mídias).
- `tutorial_artefatos.md`: roteiro completo que relaciona cada tópico da atividade aos materiais correspondentes (algoritmo, visualização, apresentação, planilhas e animações).

A pasta `imagens/` é utilizada por scripts automatizados (`scripts/gerar_midias.py`) para armazenar capturas e animações geradas.
